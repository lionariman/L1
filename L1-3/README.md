# L1-3: Работа нескольких воркеров

## Запуск:
```bash
go run main.go <количество_воркеров>
```

### Примеры:
```bash
go run main.go 5    # Запуск с 5 воркерами
go run main.go 10   # Запуск с 10 воркерами
```

## Как остановить:
`Ctrl+C` для завершения программы
